# Sport_Match

Match Sport Partner est une application mobile conçue pour les 20-35 ans qui résout le problème de l'isolement sportif en facilitant le matching de partenaires compatibles. L'application permet de trouver et créer des activités sportives en petits groupes (1 à 2 personnes) selon le niveau, l'horaire et la géolocalisation. Le cœur du challenge réside dans notre algorithme de recommandation précis et dans la construction d'une infrastructure Back-End robuste et sécurisée.
